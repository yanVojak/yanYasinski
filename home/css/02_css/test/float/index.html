Блок навигации

  1. Ширина 15em
  2. Должен присутствовать цвет фона
  3. Распологаться слева
  4. Блок навигации наполните контентом достаточным, чтобы по высоте он начинал захватывать вторую статью как показано в данном примере

Статья 1

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона

Во всех "статьях" сделайте отступ снизуот соседей равным .2em, при помощи свойства margin-bottom.
