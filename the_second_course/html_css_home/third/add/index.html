Safe pallete

All systems have what's called a system palette â€” the basic set of colours that the system can use at the 256-colour level. Sadly, Windows and Mac systems don't use the same 256 colours, but there is a cross-over of 216 colours which are in both system palettes. So, when I say the following 216 colours are non-dithering or web-safe, it means they will look the exact same on any computer. That's what makes them so useful. You can use them and be guaranteed that a user won't get a horrible looking mess when they load your page. It'll all be smooth colour.

Why are "chuknorris", "chuknorr", "CANDY" or "BruceWillis"?

Answer is simple - we need to change each letter to 16th system count or just change to 0 (if not exist such letter). after that, you need to add zeros to the string until the number of characters divide by three like this "c00c 0000 0000". As you could guess it's a color model RGB.
