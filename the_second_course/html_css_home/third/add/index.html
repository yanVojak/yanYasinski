Sadly, Windows and Mac systems don't use the same 256 colours, but there is a cross-over of 216 colours which are in both system palettes. So, when I say the following 216 colours are non-dithering or web-safe, it means they will look the exact same on any computer. That's what makes them so useful. You can use them and be guaranteed that a user won't get a horrible looking mess when they load your page. It'll all be smooth colour.
